Welcome to the Espruino Web IDE

The Espruino Web IDE is designed for writing code on microcontrollers that use the Espruino JavaScript interpreter

To help you get up to speed quickly with the IDE, this tour will run you through some of the core features you'll need to know to get started.

So when you are ready, click next to continue.

The first thing you'll want to do is connect to your Espruino board by clicking the 'Connect' button.

Connected

The terminal

To send simple commands to the Espruino, simply type them in the terminal window and hit enter to send.

Next

The code editor
